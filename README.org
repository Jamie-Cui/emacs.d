* Q & A 

** How to let MacOs hide dock?

see: https://apple.stackexchange.com/a/82084

#+begin_src bash
defaults write com.apple.dock autohide-delay -float 1000; killall Dock
#+end_src

#+begin_src bash
defaults delete com.apple.dock autohide-delay; killall Dock
#+end_src
